## 🛡️ セキュリティとデータ管理について

本システムは、利用者様の個人情報および機密性の高いケア記録を安全に扱うため、画面（フロントエンド）とデータ（バックエンド）を完全に分離したアーキテクチャを採用しています。

1. **ソースコードに個人情報を含まない（No PII in Source Code）**
   本リポジトリで公開されているコード（`index.html` 等）は、あくまでシステムを操作するための「画面（操作パネル）」です。プログラム内には、利用者様の氏名や記録などの個人情報（PII）は一切含まれていません。

2. **強固なバックエンド分離（Separation of Database）**
   実際のデータはすべて、Google の高度なセキュリティインフラで保護された「Google スプレッドシート（Google Workspace）」内に保管されています。データが GitHub 上に保存されることはありません。

3. **API経由でのアクセス（API Integration）**
   画面とデータベース間のやり取りは、Google Apps Script (GAS) を用いて構築された専用のAPIを経由して行われます。フロントエンドからは、このAPIを通じてのみデータの読み書きが実行される仕組みとなっています。
